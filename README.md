# self-graph

O self-graph é um sistema que transforma projetos reais em um grafo vivo de como um desenvolvedor pensa, projeta e evolui ao longo do tempo.

## Status do Projeto

Este projeto está em estágio inicial de desenvolvimento. A versão atual serve apenas para reserva do nome no registro NPM.

Novas versões com funcionalidades de ingestão e análise virão em breve.

## A Ideia

O objetivo é transformar a identidade técnica subjetiva em um sistema consultável e em constante evolução.

- Cada projeto torna-se um nó.
- Cada análise de LLM torna-se dado estruturado.
- Padrões técnicos emergem ao longo do tempo.

Para mais detalhes sobre o planejamento, veja o [ROADMAP.md](./ROADMAP.md).
